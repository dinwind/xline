# 如何用 Axline 提交反馈

1. 先登录 Axline 账号（Account）。
2. 点侧栏标题栏图标 **Account → Feedback（气泡）→ Settings**，或命令面板 **Axline: Feedback** / **Axline: Report Issue**；也可从 Settings → About → **Send Feedback** 进入新建表单。
3. 新建表单会显示 **Submitting as**（当前登录账号）。选择类型（Bug / Feature / Question），填写标题与描述。
4. 可附加截图、日志或文档：`png` / `jpeg` / `webp` / `gif` / `pdf` / `txt` / `md` / `log` / `json` / `csv` / `xml` / `docx`（最多 5 个，每个 ≤ **10 MB**）。
5. 「随反馈发送的环境信息」包含 Axline/主机版本、平台等 8 项，**全部必填**（不可取消）；反馈默认私密，仅你与管理员可见。
6. 提交后在 **Mine** 查看状态；团队需要更多信息时状态为 **Needs your reply**，请在详情页补充评论。
7. **Public** 页可浏览本应用已公开反馈；ACTIVE 成员可评论公开单。

状态由 AuthNexus Feedback Hub 管理；Axline 内不能改状态。附件仅经登录会话加载，不会出现在公开静态链接。
